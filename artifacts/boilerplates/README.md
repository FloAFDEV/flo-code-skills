# Boilerplates

> **Projets exécutables de démarrage** — « clone et lance ». Ils incarnent les Standards et les Patterns.

**Existant** : le starter **`flostack-saas`** (Next.js 15 + Supabase) vit à
[`../../templates/flostack-saas`](../../templates/flostack-saas). Ce dossier est le **catalogue** : il
**pointe** vers les boilerplates, il ne les héberge pas (on ne déplace pas le starter — référence, pas copie).

**Y va** : une entrée de catalogue par boilerplate (nom, périmètre, standards incarnés, lien).
**N'y va pas** : du code dupliqué depuis `/templates/`.
