# Templates (gabarits de documents)

> **Squelettes de documents à remplir** — specs, ADR, comptes rendus, etc. À ne pas confondre avec les
> **boilerplates de code** ([`../boilerplates/`](../boilerplates/)) ni avec le `/templates/` racine
> (qui héberge le code de démarrage).

**Y va** : un gabarit de fichier (Markdown/config) à compléter.
**N'y va pas** : un projet runnable (→ boilerplates), un exemple réel (→ [`../examples/`](../examples/)).

**Existant à référencer** : le gabarit d'ADR vit dans [`../../knowledge/adr/TEMPLATE.md`](../../knowledge/adr/TEMPLATE.md).
