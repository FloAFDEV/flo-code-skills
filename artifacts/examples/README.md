# Examples

> **Preuves concrètes tirées de vrais projets** — « voici, en réel ». Le matériau brut du
> [flywheel](../../knowledge/flywheel.md) : c'est à partir d'exemples qu'émergent patterns et standards.

**Y va** : un extrait **assaini** issu d'un projet réel (ex. Garage Mendonça, *externe*), daté et sourcé.
**N'y va pas** : un secret, une donnée réelle, un fichier complet de projet client (le dépôt client
n'est **jamais** modifié ni recopié).

- **Nommage** : `kebab-case.md` (ex. `garage-mendonca-rls-vehicules.md`).
- **Promotion** : un exemple qui se répète franchit la porte de maturité → devient pattern/standard.
